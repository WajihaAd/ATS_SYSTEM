# ATS_SYSTEM
Built a production-style AI Applicant Tracking System (ATS) that automates resume screening, semantic job matching, and recruiter-ready candidate ranking. The platform uses weighted ATS scoring, explainable AI insights, risk-adjusted ranking logic,and an interactive Streamlit dashboard with PostgreSQL for efficient hiring decisions
